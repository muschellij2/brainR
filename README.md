
# brainR Package:

The goal of `brainR` is to provide this includes functions for creating
3D and 4D images using ‘WebGL’, ‘rgl’, and JavaScript Commands. This
package relies on the X ToolKit
(‘XTK’,<https://github.com/xtk/X#readme>).
